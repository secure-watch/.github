# Secure Watch Organization Configuration

[한국어 문서](README.ko.md)

This repository manages Secure Watch's public organization profile and default community health files.

| Path | Purpose |
| --- | --- |
| `profile/README.md` | GitHub organization profile |
| `assets/secure-watch-logo.png` | Organization and project logo |
| `CONTRIBUTING.md` | Default contribution guidelines |
| `SECURITY.md` | Default vulnerability reporting policy |
| `SUPPORT.md` | Default support channels |
| `CODE_OF_CONDUCT.md` | Default community code of conduct |

Repository-specific policies take precedence over these defaults.
