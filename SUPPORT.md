# Support

[한국어 안내](#한국어)

| Request | Channel |
| --- | --- |
| Usage questions and general help | GitHub Discussions |
| Feature proposals and design discussions | GitHub Discussions |
| Reproducible bugs | GitHub Issues |
| Security vulnerabilities | Private vulnerability report |

When asking for help, include the version, environment, commands, expected result, and actual result. Remove tokens, internal URLs, package registry credentials, and other sensitive information.
