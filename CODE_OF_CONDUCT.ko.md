# 행동 강령

[English](CODE_OF_CONDUCT.md)

## 기본 원칙

Secure Watch 기여자는 다음을 지켜야 합니다.

- 서로 존중하고 건설적으로 소통합니다.
- 작업과 기술적 맥락에 집중합니다.
- 다양한 관점과 배경을 존중합니다.
- 근거에 기반한 피드백을 수용합니다.
- 비공개 정보와 보안 민감 정보를 보호합니다.

괴롭힘, 차별, 협박, 인신공격, 비공개 정보의 고의적 공개, 커뮤니티를 방해하는
행위는 허용하지 않습니다.

## 적용

Issue, Pull Request, Discussion 또는 댓글의 GitHub
[콘텐츠 신고](https://docs.github.com/ko/communities/maintaining-your-safety-on-github/reporting-abuse-or-spam)
기능을 사용해 행동 문제를 저장소 maintainer에게 신고합니다. 해당 기능을 사용할 수
없거나 특정 콘텐츠와 관련 없는 문제라면 GitHub Support에 신고합니다. 민감한
세부 정보는 공개 Issue나 Discussion에 작성하지 않습니다.

Maintainer는 커뮤니티를 안전하고 생산적으로 유지하기 위해 콘텐츠 삭제, 참여 제한
또는 필요한 조치를 취할 수 있습니다.
